This feature applies to Eventstream sources that use streaming connectors to retrieve data from external sources, i.e., outbound requests are made to those external systems. The following Eventstream sources are supported by this feature
- Azure SQL Database CDC
- Azure SQL Managed Instance CDC
- Azure Service Bus
- Azure Cosmos DB CDC
- Azure Data Explorer (ADX)
- Azure Event Hubs (Extended features mode)
- Amazon MSK Kafka
- Apache Kafka
- Amazon Kinesis Data Streams
- Confluent Cloud for Apache Kafka
- PostgreSQL CDC
- MongoDB CDC
- Google Cloud Pub/Sub
- MQTT
- HTTP
- Solace PubSub+
- SQL Server on VM CDC
- MySQL Database CDC
