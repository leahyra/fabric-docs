# Set up your KQL Database connection

This article outlines the steps to create a KQL Database connection.

## Supported authentication types

The KQL Database connector supports the following authentication types for copy and Dataflow Gen2 respectively.  

|Authentication type |Copy |Dataflow Gen2 |
|:---|:---|:---|
|Organizational account| √ | √ |

## Set up your connection in Dataflow Gen2

You can connect Dataflow Gen2 to KQL Database in Microsoft Fabric using Power Query connectors. Follow these steps to create your connection:

1. [Get data from Data Factory in Microsoft Fabric](/power-query/where-to-get-data#get-data-from-data-factory-in-microsoft-fabric).
1. [Set up KQL Database prerequisites](/power-query/connectors/kql-database#prerequisites).
1. [Connect to a KQL database (from Power Query Online)](/power-query/connectors/kql-database#connect-to-a-kql-database-from-power-query-online).
