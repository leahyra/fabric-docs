# Set up your Salesforce objects connection

This article outlines the steps to create a Salesforce objects connection.

## Supported authentication types

The Salesforce objects connector supports the following authentication types for copy and Dataflow Gen2 respectively.  

|Authentication type |Copy |Dataflow Gen2 |
|:---|:---|:---|
|Organizational account| √ | √ |

## Set up your connection in Dataflow Gen2

You can connect Dataflow Gen2 to Salesforce objects in Microsoft Fabric using Power Query connectors. Follow these steps to create your connection:

1. [Get data from Data Factory in Microsoft Fabric](/power-query/where-to-get-data#get-data-from-data-factory-in-microsoft-fabric).
1. [Set up Salesforce objects prerequisites](/power-query/connectors/salesforce-objects#prerequisites).
1. Check [Salesforce objects known issues and limitations](/power-query/connectors/salesforce-objects#known-issues-and-limitations) to make sure your scenario is supported.
1. [Connect to Salesforce Objects (from Power Query Online)](/power-query/connectors/salesforce-objects#connect-to-salesforce-objects-from-power-query-online).
